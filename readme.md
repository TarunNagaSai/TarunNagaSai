## Featured Projects

### Yahuah Bible Platform — *Flagship, Production*
Multi-platform Bible study system spanning web, PWA, native mobile (iOS/Android/Windows/macOS), and a production AI translation pipeline.

**Phase 3 — AI Translation Pipeline**
- Custom ETL across **31,000+ verses** using Ratcliff/Obershelp sequence matching
- DAG orchestration on RunPod with gated GPU mutex for zero-latency pipeline handoff
- HITL review system with staging DB isolation and admin publish trigger
- **60,000+ verses translated · 54,409 word replacements · 0% error rate**

**Stack:** Gemma 4 31B (full-precision) · vLLM (PagedAttention + KV cache) · RunPod RTX 6000 Blackwell 96GB · Python · Laravel 10 · MySQL · PWA · Flutter

---

### Horeb — Offline-First Flutter Bible App *(Active, Large-Scale)*
Production-grade Flutter app with strictly layered architecture (Bloc/Cubit → Service → Repository → ObjectBox/Firebase/Network).

**Architecture highlights:**
- MetaData module with single concurrency control (no double pooling)
- Isolate-based JSON conversion via `compute()` for jank-free UI
- Remote Config–driven language packs (no app update required)
- Migrated from state-heavy to data-driven → **64% Dart heap reduction**, 3× faster rendering

**Stack:** Flutter · ObjectBox · Bloc/Cubit · AutoRoute · Dio · RxDart · Firebase (Auth, FCM, Crashlytics, Remote Config, Firebase AI) · RevenueCat · Workmanager · Patrol · Fastlane
